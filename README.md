# Sitko.Core

Opinionated framework on top of .NET 8 and ASP.NET Core for building microservices. 

Provides base application and modules for fast application building.

# Modules

- Logging
- Tracing
- Service discovery
- Database access
- Search
- Queue 
- GRPC
- Storage
- Email 
- Unit testing
